# Atrium Lab 🌳
### Decentralized AI Agent Marketplace on Canopy Network

> *List, discover, and license verified AI agents onchain — every transaction transparent, every agent sovereign.*

---

## What is Atrium Lab?

Atrium Lab is an onchain AI agent marketplace built natively on the Canopy Network protocol. It enables developers to monetize their AI models and agents, while buyers can discover and purchase access licenses — all settled in CNPY with no middlemen.

---

## Core Features

- **Marketplace** — Browse and filter AI agents by category (AI Agent, Language Model, Image, DeFi, Code)
- **Deploy Agent** — Submit your AI model for listing with name, description, category and CNPY monthly price
- **Buy Access** — Purchase agent licenses in CNPY with real-time balance checks
- **My Access** — Track all active subscriptions in one dashboard
- **Wallet Connect** — Connect wallet to interact with the marketplace

---

## Canopy Template Used

TypeScript + Go (Canopy Network Templates)

## How to Run Locally

1. Clone this repo
2. Open `plugin/atrium/atrium-lab-1.html` in your browser
3. Run Canopy node locally: `canopy start --port 50002`
4. Connect wallet and interact
