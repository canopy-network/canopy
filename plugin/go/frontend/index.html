Signing Reference
BLS12-381 Signing — how this frontend does it

1. Encode inner message to protobuf bytes (manual varint encoding, field numbers from tx.proto)
2. Wrap in google.protobuf.Any with type_url = /types.Message<Type>
3. Encode Transaction{} proto with signature field absent
4. sig = bls12_381.sign(txBytes, privateKey) via @noble/curves
5. Attach signature: { publicKey, signature } as base64 and POST to /v1/tx

Get your private key:
canopy admin ks-get <nickname> --password <pwd>
Then paste the PrivateKey hex into the Signer panel above.
